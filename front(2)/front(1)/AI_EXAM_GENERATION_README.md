# AI考核生成功能说明

## 功能概述

教师端考核生成功能现在支持通过AI API智能生成考核题目，可以根据知识点权重、题目类型配置和难度要求自动生成高质量的考核内容。

## 主要功能

### 1. 考核设置
- **考核名称**：设置考核的标题
- **考核类型**：选择考核的整体类型（单选题、多选题、填空题、简答题）
- **考核时长**：设置考试时间（分钟）
- **总分**：设置考核的总分

### 2. 题目配置
- **单选题**：配置单选题数量和每题分值
- **多选题**：配置多选题数量和每题分值
- **填空题**：配置填空题数量和每题分值
- **简答题**：配置简答题数量和每题分值
- **难度设置**：每种题型都有对应的难度等级

### 3. 知识点选择
- **知识点权重**：为每个知识点设置权重（1-10）
- **知识点覆盖**：确保考核内容覆盖选定的知识点
- **智能分配**：AI根据权重自动分配题目到不同知识点

### 4. AI智能生成
- **智能出题**：AI根据配置自动生成题目
- **质量保证**：确保题目质量高、难度适中
- **知识点覆盖**：全面覆盖选定的知识点
- **格式规范**：生成标准格式的题目和答案

### 5. 生成结果展示
- **题目预览**：在线预览生成的题目
- **答案解析**：查看参考答案和详细解析
- **导出功能**：支持导出PDF格式
- **保存功能**：保存到考试库

## 使用流程

### 1. 设置考核基本信息
1. 输入考核名称
2. 选择考核类型
3. 设置考核时长和总分

### 2. 配置题目类型
1. 为每种题型设置数量
2. 系统自动计算总分
3. 确保题目数量合理

### 3. 选择知识点
1. 勾选需要考核的知识点
2. 为每个知识点设置权重
3. 权重影响题目分配

### 4. 生成考核
1. 点击"智能生成考核"按钮
2. AI根据配置生成题目
3. 等待生成完成

### 5. 查看和保存
1. 预览生成的题目
2. 检查题目质量和覆盖度
3. 导出PDF或保存到系统

## 技术实现

### 前端技术
- Vue 3 + Element Plus
- 响应式表单设计
- 实时计算和验证
- PDF导出功能

### 后端技术
- Spring Boot 3
- AI接口集成
- JSON解析和格式化
- 题目数据结构化

### API接口
- `POST /api/ai/generate-exam` - AI考核生成
- `POST /api/exam/create` - 保存考核
- `GET /api/exam/{id}` - 获取考核详情

## 数据结构

### 请求数据格式
```json
{
  "subject": "计算机科学",
  "difficulty": "中等",
  "examName": "期末考试",
  "questionCount": 20,
  "duration": 120,
  "totalScore": 100,
  "knowledgePoints": [
    {
      "id": 1,
      "name": "基础概念",
      "weight": 5
    }
  ],
  "questionTypes": [
    {
      "type": "choice",
      "count": 10,
      "scorePer": 2,
      "difficulty": "简单"
    }
  ]

```

## 题目类型说明

### 单选题 (choice)
- 提供4个选项
- 只有一个正确答案
- 适合考查基础概念

### 多选题 (multiple)
- 提供4个选项
- 可能有多个正确答案
- 适合考查综合理解

### 填空题 (fill)
- 需要填写答案
- 答案相对固定
- 适合考查记忆和理解

### 简答题 (essay)
- 需要详细回答
- 答案相对开放
- 适合考查分析和应用

## 注意事项

1. **知识点选择**：至少选择一个知识点
2. **题目数量**：确保总题目数大于0
3. **分数分配**：注意总分与题目数量的匹配
4. **网络连接**：需要稳定的网络连接调用AI接口
5. **生成时间**：AI生成可能需要一些时间，请耐心等待
6. **内容质量**：生成的内容质量取决于配置的合理性
