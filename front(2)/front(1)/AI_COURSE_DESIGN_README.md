# AI教案生成功能说明

## 功能概述

教师端AI备课助手现在支持直接连接API生成PPT和教案，并提供文件上传功能，可以根据上传的已有文件生成相应的教学资源。

## 主要功能

### 1. 文件上传功能
- 支持上传多种格式文件：doc、docx、pdf、txt、md、ppt、pptx
- 单个文件大小限制：10MB
- 支持拖拽上传和点击上传
- 支持多文件同时上传

### 2. AI生成选项
- **PPT课件生成**：自动生成课程PPT，包含标题页、大纲页、内容页等
- **教案生成**：生成完整的课程教案，包含教学目标、教学内容、教学方法等
- **教学内容设计**：设计详细的教学内容和时间安排
- **实训练习设计**：设计相关的实践练习和作业
- **时间分布规划**：规划课程的时间安排和进度
- **考核题目生成**：生成相关的考核题目和答案
- **教学建议**：提供教学方法和策略建议

### 3. 生成结果展示
- **PPT预览**：在线预览生成的PPT内容
- **教案预览**：查看生成的教案内容
- **内容展示**：展示教学内容、练习、考核等详细信息
- **下载功能**：支持下载PPT和教案文件

## 使用流程

### 1. 上传文件（可选）
1. 在"上传已有文件"区域拖拽或选择文件
2. 点击"开始上传"按钮上传文件
3. 上传成功后，文件信息会显示在列表中

### 2. 填写课程信息
1. 输入课程名称、类型、时长、难度等基本信息
2. 填写课程大纲和特殊要求
3. 如果上传了文档文件，系统会自动提取内容作为课程大纲

### 3. 选择生成选项
1. 勾选需要生成的内容类型
2. 建议选择"PPT课件"和"教案"选项
3. 可以根据需要选择其他选项

### 4. 生成课程设计
1. 点击"生成课程设计"按钮
2. 系统会调用AI接口生成相应的教学资源
3. 生成完成后会显示在结果区域

### 5. 查看和下载
1. 在结果区域查看生成的内容
2. 可以下载PPT和教案文件
3. 可以保存设计结果

## 技术实现

### 前端技术
- Vue 3 + Element Plus
- 文件上传组件
- PPT生成库：pptxgenjs
- Word文档生成：docx

### 后端技术
- Spring Boot 3
- 文件上传处理
- AI接口集成
- JSON解析和格式化

### API接口
- `POST /api/ai/upload/file` - 文件上传
- `GET /api/ai/download/{fileName}` - 文件下载
- `POST /api/ai/course-design` - 课程设计生成

## 注意事项

1. **文件格式**：只支持指定的文件格式，其他格式会被拒绝
2. **文件大小**：单个文件不能超过10MB
3. **网络连接**：需要稳定的网络连接来调用AI接口
4. **生成时间**：AI生成可能需要一些时间，请耐心等待
5. **内容质量**：生成的内容质量取决于输入的课程信息和AI模型
